# Project Summary
The project is focused on developing a gamified sales funnel application, utilizing modern web technologies to enhance user engagement and track progress through various challenges. The application integrates with Supabase for real-time data management, allowing users to navigate a series of interactive tasks designed to improve their emotional intelligence and decision-making skills.

# Project Module Description
- **Gamification Engine**: Implements a series of challenges that users must complete, each designed to enhance specific skills or knowledge areas.
- **Checkpoint System**: Allows users to save their progress and return to their last completed challenge seamlessly.
- **User Progress Tracking**: Utilizes Supabase to monitor user interactions and progress through the application, capturing valuable data for analysis.
- **Dynamic Pricing and Analytics**: Adjusts user experiences based on performance metrics and user data.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Authentication)
- **Build Tool**: Vite
- **State Management**: Custom hooks and context

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the application:
   ```bash
   npm run build
   ```
3. Run the application locally:
   ```bash
   npm run dev
   ```
